# Weather App 🌤️

A responsive weather application that provides real-time weather data using a public API.

## Features
- Search weather by city
- Real-time data (temperature, humidity, wind)
- Responsive design
- Error handling

## Tech Stack
- HTML
- CSS
- JavaScript
- API (OpenWeather)
- Netlify
